# KinetiPass - Advanced AI Biometric Proctoring

KinetiPass is a privacy-first, zero-invasion academic integrity system. It verifies student identities during online examinations using real-time keystroke dynamics (dwell times and flight times) rather than intrusive webcams, microphones, or screen recordings.

## Features

- **Zero-Invasion Privacy**: Authenticates typing rhythm without webcams, audio, or screen capture.
- **Low-Bandwidth Friendly**: Operates efficiently on low-connectivity networks (e.g. 2G) as keystroke metrics only require a few bytes.
- **Anxiety-Free Assessments**: Removes webcam-induced anxiety, preserving students' mental bandwidth and performance.
- **Interactive Simulator**: Includes a student exam editor with real-time biometric analysis and warning visualizer.
- **Admin Dashboard**: Live dashboard tracking typing rhythm consistency, verifying identity scores, and flagging anomalies.
